# App Development Planning

## Planning Template
1. Problem and target users.
2. Contract changes (proto/http/config).
3. Core data flow and dependency map.
4. Middleware/interceptor impacts.
5. Infra and deployment impacts (ArgoCD/Terraform/CI).
6. Risk + rollback.
7. Validation and observability plan.

## Implementation Checklist
1. API contract finalized.
2. Data/storage changes reviewed.
3. Authorization and rate-limiting considered.
4. CI tests include all required services.
5. Dashboards/alerts updated if required.
6. Runbook/incident docs updated.
